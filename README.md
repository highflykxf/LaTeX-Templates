# LaTeX-Templates

整理一些自己用到的LaTeX模板。

## 仓库结构说明

```
|-homework.tex  % 作业模板
|-ppt.tex       % ppt模板
|-resume.tex    % 简历模板
|-preview       % 预览图
|-thesis
    |-thu       % 清华大学学位论文模版
    |-buaa      % 北京航空航天大学学位论文模版
```
